== Description ==

Wordpress project source code download

In order to configure settings, navigate to **[ Wordpress Admin -> Settings -> Create download ZIP ]**, where you can see options:

project source code download


= NOTE =


We have this plugin compatible gutenberg.

== Installation ==

1. Upload `project-source-code-download` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin from Admin > Plugins menu.
3. Once activated you should check with Settings > Create download ZIP
4. Use project source code download.

Make sure you get the latest version.
